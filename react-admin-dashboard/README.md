# Onyx Admin Dashboard

A comprehensive, minimalist, and responsive Admin Dashboard built to manage internal operations, track analytics, and monitor data efficiently.

##  Technologies Used

- *React.js*: For building the dynamic user interface.
- *Tailwind CSS*: For a clean, fully responsive, and utility-first styling.
- *Recharts*: For interactive data visualization and mock analytics.
- *Framer Motion*: For smooth and professional UI animations.
